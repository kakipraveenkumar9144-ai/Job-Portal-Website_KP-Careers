KP Careers – Job Portal Website
KP Careers is a modern and responsive job portal application developed using HTML, CSS, and JavaScript. The platform provides a simple and user-friendly interface for job seekers to browse available jobs, view company details, and apply for suitable positions.

Features
Responsive design for mobile and desktop devices

Job listings with detailed information

Search and filter functionality

User-friendly interface

Apply for jobs with a single click

Clean and modern UI

Technologies Used
HTML5

CSS3

JavaScript

Project Purpose
This project was developed to strengthen front-end development skills and demonstrate the ability to create real-world web applications for a Python Full Stack Developer portfolio.
